# Kleiderspenden-Portal

Dies ist eine Webanwendung zur Registrierung von Kleiderspenden für einen gemeinnützigen Verein.

## Funktionen

- Registrierung einer Kleiderspende
- Auswahl zwischen Übergabe an der Geschäftsstelle und Abholung
- Auswahl der Kleidungsart
- Auswahl eines Krisengebiets
- Prüfung der Postleitzahl bei Abholung
- Bestätigungsanzeige nach erfolgreicher Registrierung
- Responsive Design mit Bootstrap

## Technologien

- HTML
- CSS
- JavaScript
- Bootstrap 5

## Geschäftsstellen-PLZ

Die Geschäftsstelle besitzt beispielhaft die PLZ 12345.  
Abholungen sind nur möglich, wenn die ersten zwei Ziffern der Abhol-PLZ ebenfalls `12` sind.

Header
- Logo
- Titel
- Navigation

Content
- kurze Beschreibung
- Formular zur Registrierung

Footer
- rechtliche Hinweise
- Impressum-Hinweis
- Datenschutz-Hinweis

## Formularstruktur 

Spendenart:
- Übergabe an der Geschäftsstelle
- Abholung

Kleidungsart:
- Jacken
- Hosen
- Pullover
- Schuhe
- Kinderkleidung
- Sonstiges

Krisengebiet:
- Ukraine
- Gaza
- Sudan
- Syrien
- Haiti

Bei Abholung zusätzlich:
- Name
- Straße
- Hausnummer
- Postleitzahl
- Ort
